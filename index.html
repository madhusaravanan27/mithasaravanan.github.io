Real Time Sports Analytic Platform

Project Overview

The Real Time Sports Analytic Platform is a distributed data pipeline designed to process live lacrosse game streams, update player and team statistics, and generate live box scores. The project integrates PySpark, MongoDB, MSSQL, and Minio to simulate a real-world sports analytics environment.

Technologies Used

  • PySpark: For distributed data processing.
  • MongoDB: Stores real-time box scores.
  • Microsoft SQL Server (MSSQL): For team and player reference data.
  • Minio: S3-compatible storage for game stream data.
  • Apache Drill: SQL query engine for data integration.
  • Jupyter Lab: Used for developing PySpark code.

Key Accomplishments

  • Real-Time Data Stream Processing: Developed a PySpark pipeline to ingest and track real-time game events from Minio.
  • Box Score Generation: Generated live JSON-formatted box scores, updated regularly in MongoDB for web access.
  • Player and Team Statistics Update: Updated final team and player stats in MSSQL using PySpark for cumulative calculations.
  • Database Integration and Querying: Queried integrated data from MSSQL, MongoDB, and Minio using Apache Drill.
  • End-to-End Data Pipeline: Automated data ingestion, transformation, and real-time statistics updates, providing up-to-date insights throughout the game.
